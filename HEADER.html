Current Version

    Currently implemented:

  • Tabbed switching between four schedules.
  • Nearly full cascade style searching. (Instructor, Department, Course Number, Start time, Core Fulfillment).
  • Two seperate and independent search menus.
  • Tutorial is in and accessible from the top right of the screen.
  • Adding classes to multiple schedules using the checkbox's associated with each schedule results.
  • Hitting the print button puts the current schedule into a printable version and opens the built in browser print.
  • All times are displayed according to the international standard notation of time: 24-hour clock.
  • Classes are now removed from the schedule by clicking a red X.

    Not yet implemented:

  • Possibly redo the "checkbox" idea to something more intuitive.

    Current bugs:

  • At some point after adding multiple classes, subsequent classes will cease to be in any color.
  • I was able to add classes to the schedule in a way that did not cycle through colors correctly and ran out of colors. I was also able to have three classes on a schedule with two of them in the same color. (AB)
  • The department listing is not in alphabetical order like cascade.
